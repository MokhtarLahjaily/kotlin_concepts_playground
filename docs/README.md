## Exécuter les démos (PowerShell)

Dans le dossier du projet:

```
./gradlew runLauncher
./gradlew runVariablesTypesDemo
./gradlew runNullSafetyDemo
./gradlew runCollectionsDemo
./gradlew runControlFlowDemo
./gradlew runFunctionsLambdasDemo
./gradlew runSequencesDemo
./gradlew runOOPDemo
./gradlew runExtensionsDataEnumDemo
./gradlew runOperatorOverloadingDemo
./gradlew runScopeFunctionsDemo
```

Astuce: `-q` pour sortie concise, `--no-daemon` pour éviter le daemon.
